# About the Mara

reference to [wikipedia] (https://en.wikipedia.org/wiki/Mara_(mammal) )

## Important facts:
1.  Mammalia family.
2.  relatives of guinea pigs.
3.  may amble, hop in a rabbit-like fashion, gallop, or bounce on all fours.

## Lives in:
- South America
